# Thư viện chuẩn / Standard Library

!!! warning "Đang phát triển / Under Development"
    Thư viện chuẩn đang được xây dựng. Hiện tại chỉ có `in_ra` khả dụng.

---

## io — Nhập/Xuất / Input/Output

| Hàm | Mô tả | Trạng thái |
|---|---|---|
| `in_ra(giá_trị)` | In giá trị ra màn hình | ✅ |
| `đọc_dòng()` | Đọc một dòng từ bàn phím | 📋 |
| `đọc_tệp(đường_dẫn)` | Đọc nội dung file | 📋 |
| `ghi_tệp(đường_dẫn, nội_dung)` | Ghi nội dung vào file | 📋 |

## toán — Toán học / Math

| Hàm | Mô tả | Trạng thái |
|---|---|---|
| `căn_bậc_hai(x)` | Căn bậc hai | 📋 |
| `lũy_thừa(x, n)` | x mũ n | 📋 |
| `tuyệt_đối(x)` | Giá trị tuyệt đối | 📋 |
| `tối_đa(a, b)` | Giá trị lớn nhất | 📋 |
| `tối_thiểu(a, b)` | Giá trị nhỏ nhất | 📋 |

## chuỗi — Chuỗi ký tự / Strings

| Hàm | Mô tả | Trạng thái |
|---|---|---|
| `dài(chuỗi)` | Độ dài chuỗi | 📋 |
| `nối(a, b)` | Ghép hai chuỗi | 📋 |
| `cắt(chuỗi, từ, đến)` | Cắt chuỗi con | 📋 |
| `chứa(chuỗi, tìm)` | Kiểm tra chuỗi con | 📋 |

## hệ_thống — Hệ thống / System

| Hàm | Mô tả | Trạng thái |
|---|---|---|
| `thoát(mã)` | Kết thúc chương trình | 📋 |
| `biến_môi_trường(tên)` | Đọc biến môi trường | 📋 |
| `đối_số()` | Đọc tham số dòng lệnh | 📋 |
