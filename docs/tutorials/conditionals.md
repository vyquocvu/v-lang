## Toán tử so sánh / Comparison Operators

| Toán tử | Ý nghĩa |
|---|---|
| `==` | Bằng |
| `!=` | Khác |
| `<` | Nhỏ hơn |
| `>` | Lớn hơn |
| `<=` | Nhỏ hơn hoặc bằng |
| `>=` | Lớn hơn hoặc bằng |
